## 🎯 QUICK START

### 1. Install Dependencies
```bash
npm install
```
*(Already done, but run this after pulling changes)*

### 2. Start Development Server
```bash
npm run dev
```
- Opens http://localhost:3000 automatically
- HMR enabled for instant updates
- All changes reflected immediately

### 3. Run Your Code
- Write M68K assembly in the editor
- Click "Run" to execute
- Monitor registers and memory
- Click "Step" for step-by-step debugging

### 4. Build for Production
```bash
npm run build
```
- Creates optimized `dist/` folder
- Ready to deploy anywhere
- Can be deployed to GitHub Pages with `npm run deploy`
